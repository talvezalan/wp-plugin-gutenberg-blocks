# Changelog

## [1.3.0] - 2026-01-03
### Agregado
- **Woo Pop-out Cards**: Nuevo bloque para mostrar productos de WooCommerce con diseño minimalista y efecto pop-out.
- **Woo Pop-out Cards**: Botones de navegación en móvil y soporte para hasta 6 productos.
- **Woo Pop-out Cards**: Integración con Schema.org Product.

## [1.2.0] - 2025-12-30
### Agregado
- **Flexible Link Carousel**: Nuevo bloque versátil para mostrar categorías o logos con diseño de carrusel.

### Corregido
- **Hero Carousel Fullwidth**: Se agregó `acfb-z-0` al contenedor principal para crear un nuevo contexto de apilamiento y evitar que el contenido del bloque cubra los menús desplegables del sitio.

## [1.1.2] - 2025-12-20
### Agregado
- **Hero Carousel Fullwidth**: Soporte para enlaces en slides (`slide_link`).
